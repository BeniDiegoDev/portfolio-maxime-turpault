**🎨 Full CSS (Pas de bootstrap utilisé)**

## Comment l'utiliser

Cloner ce repo. Vous aurez besoin de `node.js` et `git` installé sur votre machine.

## Installation et Mise en place

1. Installer : `npm install`

2. Lancer : `npm start`

Une page se lancera automatiquement
